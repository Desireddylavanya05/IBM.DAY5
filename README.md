# IBM.DAY5
Library Book Management System (Frontend)

A simple **Library Book Management System frontend** built using **React** and **Vite**.
This project demonstrates a clean UI structure, component-based design, and modern frontend tooling.

---

 Features

*  Display library books
*  Built with **React 19**
*  Fast development using **Vite**
*  Simple and clean UI
*  Reusable components
*  ESLint configured for clean code

---

 Tech Stack

Frontend

* **React** `^19.2.0`
* **React DOM** `^19.2.0`

 Build & Tooling

* **Vite** `^7.2.4`
* **ESLint** `^9.39.1`
* **@vitejs/plugin-react**
* **TypeScript type support** for React

 Getting Started

Clone the Repository

```bash
git clone https://github.com/your-username/library-frontend.git
```

 Navigate to the Project Folder

```bash
cd library-frontend
```

 Install Dependencies

```bash
npm install
```

> This uses `package-lock.json` to install exact dependency versions.

 Start Development Server

```bash
npm run dev
```

Application will run at:

```
http://localhost:5173
```



 Available Scripts
